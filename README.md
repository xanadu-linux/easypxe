Script to install all the software for pxe boot on Ubuntu

## Install ##

Like root user:

```bash
git clone https://github.com/sinfallas/easypxe.git
cd easypxe
chmod +x easypxe
./easypxe
```

After run the script change the ip to 192.168.1.1
